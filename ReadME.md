
# ReadMe Generator #

![Gif](utils/readme.gif)

## About The Page ## 

Creating a command-line application that dynamically generates a README.md from a user's input. The application will be invoked with the following command:

```sh
node index.js
```

The user will be prompted for their GitHub username and other information pertaining to the project the README is for.
